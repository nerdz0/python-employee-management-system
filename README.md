# Enterprise Employee Management System (EMS)

A high-performance, full-stack employee management portal built with a modern hybrid architecture.

## 🚀 Tech Stack
- **Frontend:** Next.js 14 (App Router), Tailwind CSS, Lucide Icons
- **Backend:** FastAPI (Python 3.11)
- **Deployment:** Vercel (Serverless Functions)
- **API Documentation:** Swagger UI (Auto-generated)

## 🛠️ Key Features
- **Hybrid Architecture:** Uses Next.js for a lightning-fast UI and FastAPI for robust data processing.
- **Dynamic Directory:** Fetches employee records from a Python-based REST API.
- **Enterprise UI:** Professional dashboard layout with real-time stats and responsive design.
- **Serverless Ready:** Optimized for zero-config deployment on Vercel.

## 📂 Project Structure
- `employee-management-system/src/app`: Frontend UI and Layout.
- `employee-management-system/api`: Python FastAPI backend logic.
- `employee-management-system/vercel.json`: Deployment configuration for routing.
